# Implement Screen

## Inputs

Required

- Specification JSON

Optional

- Screen ID override
- Notes

## Workflow

1. Read prototype-constitution.md
2. Read project-config.md
3. Read specification
4. Extract screenId
5. Extract screenNodeId
6. Open corresponding Figma frame
7. Read routing from specification
8. Reuse existing components
9. Implement only this screen
10. Run Visual QA

Do not invent routing or node IDs.
